## PAGE SPECIFICATIONS

### HOME PAGE (/)

**Purpose:** Hook visitors, communicate core message, drive action

**Sections:**

1. **Hero Section**
   - Headline: "Technology Should Serve Humanity"
   - Subhead: "A Complete Political Philosophy for the Automation Age"
   - CTA Button: "Read the Manifesto"
   - Background: Abstract imagery suggesting human-tech harmony

2. **The Challenge**
   - Brief (50 words) statement of the problem
   - Key statistic (47% of jobs at risk)
   - Compelling question

3. **Our Solution** (3-column layout)
   - Universal Basic Income
   - Open Source AI Governance
   - Shared Prosperity
   - Each with icon, one-line description, "Learn more" link

4. **Core Principles** (horizontal scroll or carousel)
   - The Five Laws of Ethical AI
   - Brief visual representation

5. **Get Involved CTA**
   - "Join the Movement"
   - Newsletter signup
   - Chapter finder

6. **Latest News/Blog**
   - 3 most recent posts

7. **Footer**
   - Navigation links
   - Social media
   - Contact info
   - Legal links

### POLICIES SECTION

Each policy page follows consistent structure:

#### Template for /policies/[policy-name]

1. **Header**
   - Policy name
   - One-sentence summary
   - "Download One-Pager" button

2. **The Problem**
   - 2-3 paragraphs
   - Key statistics
   - Human stories if available

3. **Our Solution**
   - Clear explanation
   - Key features
   - Comparison to alternatives

4. **How It Works**
   - Step-by-step or visual diagram
   - Implementation details
   - Timeline if applicable

5. **Who Benefits**
   - Breakdown by demographic
   - Concrete examples
   - Calculator tool if possible (especially for Citizen's Royalty)

6. **Addressing Concerns**
   - Common objections
   - Evidence-based responses
   - Links to deeper research

7. **Related Policies**
   - How this connects to other proposals
   - Links to related pages

8. **Take Action**
   - CTA appropriate to policy
   - Share buttons
   - Newsletter signup
