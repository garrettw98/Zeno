**Post 2 - Land Value Tax Explained**
```
HOOK: "Land Value Tax: The policy that even libertarians love."

BODY: "Simple idea: Tax land, not buildings.

Why? Land is fixed - you can't create more. When land values rise, it's
usually because the COMMUNITY improved - better schools, transit, jobs.

Current system: Tax buildings = discourage construction.
LVT: Tax land = encourage construction, discourage speculation.

Benefits:
- Can't evade (land can't move)
- Progressive (wealthy own more land)
- Encourages productive use
- Captures community-created value

Henry George figured this out in the 1800s. We're still catching up."
```
